Definition of a Limit

The limit of a function as x approaches a certain value, c, is the value that
the function approaches or tends to as x gets arbitrarily close to c.
It represents the behavior of the function near that particular point.

limx→1+ (x-1) = (1.00...1-1) = 0.00...1 = 0

limx→1- (x-1) = (0.999...-1) = -0.00...1 = 0

In calculus, a limit captures the behavior of a function as the input approaches
a specific value.
One-sided limits describe the behavior as the input approaches
from either the left or the right side.For example, consider the function \(f(x) = x-1\)
as \(x\) approaches 1. The limit from the right side (\(\lim_{{x\to 1^+}}\)) and the limit from the
left side (\(\lim_{{x\to 1^-}}\)) both approach 0.When the left-sided and right-sided limits
are equal, a limit exists at that point.
In this case, since \(\lim_{{x\to 1^+}} (x-1) = \lim_{{x\to 1^-}} (x-1) = 0\), the limit \(\lim_{{x\to 1}} (x-1)\)
is 0, indicating that the function \(f(x) = x-1\) approaches 0 as \(x\) approaches 1.
